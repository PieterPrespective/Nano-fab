# Levels

Each puzzle is a data-driven JSON level definition — no code per level.
The schema lives in `src/engine/levels.ts` (validated at load time) and is
documented in `prompts/nf01/03-engine-levels-scoring.md` and `CLAUDE.md`.

Phase 1 ships levels `l1-01.json` … `l1-06.json` (Layer 1 device puzzles).
